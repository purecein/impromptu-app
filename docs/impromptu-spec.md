# Impromptu — 기능 명세서

## 앱 개요

즉흥 연주나 아이디어를 놓치지 않고 빠르게 MIDI로 캡처하는 macOS 메뉴바 앱.
레코딩된 MIDI 파일은 Logic Pro 등 DAW에서 바로 활용 가능.

---

## 기술 스택

| 항목 | 선택 |
|------|------|
| 언어 | Swift |
| UI | SwiftUI |
| MIDI | CoreMIDI |
| 오디오 엔진 | AVAudioEngine + AVAudioUnitSampler |
| 사운드폰트 | SF2 포맷 |
| 빌드 | Xcode + Swift Package Manager |
| 대상 플랫폼 | macOS — Universal Binary (Intel + Apple Silicon) |
| 최소 지원 버전 | macOS 13 Ventura |

---

## 사운드폰트

앱 번들에 사운드폰트를 포함하지 않음. 설정 > 사운드폰트 탭에서 사용자가 직접 다운로드.

### 다운로드 가능 사운드폰트

**GeneralUser GS** (기본 권장)
- GM 128종 악기 전체 포함
- 용량 약 30MB
- 라이선스: 무료 (상업 배포 가능)

**Salamander Grand Piano** (고품질 피아노)
- Yamaha C5 샘플 기반, 벨로시티 레이어 16단계
- 용량 약 800MB ~ 1GB
- 라이선스: CC BY 3.0 (출처 표기 후 상업 배포 가능)
- 이미 변환된 SF2 배포본 사용 (직접 패키징 불필요)

## MIDI 스펙

- MIDI 1.0 표준 기반
- 벨로시티: 0 ~ 127 (128단계) 완전 수신 및 저장
- 저장 포맷: SMF Format 0 (단일 트랙, DAW 호환성 최우선)
- 틱 해상도: 480 PPQN (Logic Pro 기본값과 동일)
- 템포 트랙: 저장 시 사용자 지정 BPM으로 저장 (기본 BPM은 설정에서 변경 가능, 초기값 120 BPM)
- 지원 이벤트:
  - Note On / Note Off
  - 벨로시티
  - 피치벤드
  - 모듈레이션 (CC1)
  - 채널 애프터터치
  - 서스테인 페달 (CC64)
  - Program Change (악기 번호)
  - 기타 CC 이벤트
- 저장 시 파일 첫 부분에 아래 메타 이벤트 삽입:
  - `Meta Event 0x04 (Instrument Name)`: 선택된 GM 악기명 (예: `Acoustic Grand Piano`)
  - `Meta Event 0x01 (Text)`: 선택된 사운드폰트명 (예: `SF2:GeneralUser GS`)
  - DAW에서 열 때 메타 텍스트는 무시되며 재생에 영향 없음
  - 앱 내 재생 시 메타 이벤트를 읽어 저장 당시 사운드폰트 및 악기 자동 복원

## 핵심 기능

### 1. MIDI 입력
- USB 연결 MIDI 기기 자동 인식
- 기기 연결 / 해제 실시간 감지
- 복수 기기 동시 연결 및 동시 레코딩 지원
- 복수 기기 레코딩 시 기기별로 별도 파일 저장
  - 기기명은 CoreMIDI에서 자동으로 읽어와 정규화 (예: `KOMPLETE KONTROL S61` → `S61`)
  - 단일 기기 연결 시에는 기기명 없이 기존 규칙 유지

### 2. 레코딩 트리거
다음 방식 중 설정에서 선택 가능 (복수 지정 가능):

| 방식 | 설명 |
|------|------|
| MIDI 버튼 | MIDI 기기의 특정 버튼 지정 (예: S61 Transport 버튼) |
| 키보드 단축키 | macOS 키보드 단축키 (예: Cmd+R) |
| 특정 MIDI 노트 | 건반의 특정 키를 트리거로 지정 |
| 앱 UI 버튼 | 메인 윈도우 또는 메뉴바 버튼 클릭 |

#### 시작 / 종료 트리거 별도 지정
- 레코딩 시작 트리거와 종료 트리거를 각각 다른 버튼 / 노트로 지정 가능
- 입력 장치별로 트리거를 개별 지정 가능 (예: S61은 PLAY 버튼으로 시작, STOP 버튼으로 종료)

#### MIDI Learn 방식 (MIDI 버튼 / 특정 MIDI 노트 공통)
설정에서 트리거를 지정할 때 MIDI Learn 방식을 사용:

1. 설정에서 "트리거 감지" 버튼 클릭
2. 앱이 대기 상태로 전환 ("원하는 건반 또는 버튼을 누르세요" 안내 표시)
3. 사용자가 원하는 건반 또는 MIDI 버튼 누름
4. 해당 MIDI 노트 / CC 번호 자동 감지 및 저장
5. MIDI 번호를 직접 입력할 필요 없음

#### 트리거 효과음
- 레코딩 시작 / 종료 시 macOS 기본 시스템 사운드 재생 여부 설정에서 선택 가능
- 기본값: 효과음 켜짐

#### 트리거 노트 충돌 처리
- 시작 트리거 노트 / 종료 트리거 노트 모두 레코딩 데이터에서 제외
- 종료 트리거 노트 포함 시 Note Off 없는 불완전한 이벤트가 생성되므로 제외 처리

> **사전 검증 완료**: KOMPLETE S61 Transport 버튼(PLAY, REC, STOP 등)이 DAW 없이 독립적으로 MIDI 신호를 전송하는 것을 MIDI Monitor로 확인함.

### 3. 레코딩
- Note On/Off, 벨로시티, CC 이벤트, 타임스탬프 기록
- 자유 템포 컨셉 (메트로놈 없음)
- 레코딩 중 시각적 상태 표시 (메뉴바 아이콘 및 메인 윈도우)
- 활성화된 MIDI 입력 장치가 없을 경우 레코딩 시작 버튼 비활성화

### 4. 저장 및 파일 관리
- 레코딩 종료 후 저장 방식은 설정에서 선택:
  - **다이얼로그 방식**: 레코딩 종료 후 BPM 입력 다이얼로그 표시 (기본값은 설정에서 지정한 BPM)
  - **자동 저장 방식**: 설정에서 지정한 기본 BPM으로 즉시 저장, 흐름 유지
- 다이얼로그 방식에서 취소 시:
  - 레코딩 데이터를 임시 보관, 스튜디오 목록에 [미저장] 상태로 표시
  - 미저장 항목에서 [저장] 버튼 클릭 시 BPM 입력 다이얼로그 재표시
  - 앱 종료 시 미저장 데이터는 삭제됨
- 저장된 파일 목록에서 BPM 편집 가능 (파일 재저장)
- MIDI 파일 (.mid) 로 저장
- 파일명 규칙:
  - 단일 기기: `Impromptu_YYMMDD_HHMMSS.mid` (예: `Impromptu_260409_143022.mid`)
  - 복수 기기: `Impromptu_YYMMDD_HHMMSS_기기명.mid` (예: `Impromptu_260409_143022_S61.mid`)
- 표준 MIDI 포맷으로 Logic Pro, Ableton, GarageBand 등 모든 DAW와 호환
- 저장된 파일 목록 UI
- 파일 삭제 시 확인 다이얼로그 표시
- 저장 경로 설정 가능
- Finder에서 열기 지원 (우클릭 메뉴 또는 버튼)

### 5. 재생
- 저장된 MIDI 파일 앱 내에서 바로 재생
- 파일 내 메타 이벤트를 읽어 저장 당시 사운드폰트 및 악기 자동 복원하여 재생
- 해당 사운드폰트가 미설치 상태일 경우 DLS로 폴백 재생

### 7. 악보 뷰어
- 저장된 MIDI 파일을 스케치 수준의 악보로 표시
- BPM 기반 양자화로 음표 길이 판별 (4분, 8분, 16분, 점음표)
- 쉼표 기본 처리
- 박자표 4/4 고정, 조표 C장조 기본값
- 화음 수직 표시
- 렌더링 엔진: VexFlow (MIT 라이선스) + WKWebView
- PDF 내보내기 지원 (macOS 기본 API 활용)


- 건반 누를 때 즉시 사운드 재생
- 선택된 사운드폰트 및 악기 번호로 출력
- 악기 변경 시 즉시 반영
- 사운드폰트 미설치 시 macOS 내장 DLS로 폴백 재생 (음질 낮음)

## UI

### 메뉴바
- 메뉴바 상주 (별도 Dock 아이콘 없음)
- 레코딩 상태 아이콘 표시
- 빠른 레코딩 시작 / 종료 접근
- 활성화된 MIDI 입력 장치 없을 시 "레코딩 시작" 메뉴 항목 비활성화 + 툴팁 표시
  - "MIDI 입력 장치를 설정에서 선택해주세요"

```
┌─────────────────────────────────────┐
│ Impromptu  [대기 중]                 │
├─────────────────────────────────────┤
│ 레코딩 시작 (비활성화)          ⌘R  │
│ ⚠ MIDI 입력 장치를 설정에서         │
│   선택해주세요                       │
│ 스튜디오 열기                   ⌘O  │
│ 설정                            ⌘,  │
├─────────────────────────────────────┤

└─────────────────────────────────────┘
```

### 스튜디오

- 하위 영역: 저장된 파일 목록, 파일 재생 / 악보 / BPM 편집 / 삭제 / Finder에서 열기

| 사운드폰트 | 사운드폰트 관리 | 기본 번들 확인 및 추가 사운드폰트 다운로드 |

## 추후 확장 아이디어

현재 스펙에는 포함되지 않지만 향후 고려해볼 수 있는 기능들.

### 노브 활용
- S61의 8개 로터리 노브를 MIDI CC로 매핑하여 앱 기능과 연동
  - 예: 노브 1 → 재생 볼륨 조절
  - 예: 노브 2 → 악기 변경

### 노브로 악기 변경 + 다중 채널 레코딩
- 레코딩 중 노브 또는 버튼으로 악기를 전환하면 MIDI 채널이 함께 전환되어 저장
- 채널별로 다른 악기가 지정된 상태로 하나의 MIDI 파일에 저장 (채널 1 = 피아노, 채널 2 = 스트링 등)
- DAW에서 열면 채널별로 분리하여 편집 가능
- 또는 악기별로 별도 파일로 저장하는 방식도 고려 가능
  - 예: `Impromptu_260409_143022_Piano.mid`, `Impromptu_260409_143022_Strings.mid`

## 개발 단계 계획

| 단계 | 내용 |
|------|------|
| 0 | KOMPLETE S61 Transport 버튼 MIDI 신호 여부 사전 검증 |
| 1 | 메뉴바 앱 기본 틀 + MIDI 기기 연결 확인 |
| 2 | AVAudioEngine + SF2 실시간 사운드 재생 (레코딩 테스트 전 사운드 먼저) |
| 3 | 레코딩 시작 / 종료 트리거 + 이벤트 저장 |
| 4 | MIDI 파일 저장 (Format 0, 480 PPQN, 저장 시 BPM 지정) |
| 5 | 저장 파일 목록 UI + 재생 |
| 6 | 악기 변경 + 사운드폰트 선택 |
| 7 | 설정 (MIDI 장치, 오디오 출력, 트리거 방식) |
| 8 | 사운드폰트 다운로드 지원 (GeneralUser GS + Salamander Grand Piano) |
| 9 | 악보 뷰어 (VexFlow 기반, BPM 양자화, PDF 내보내기) |
